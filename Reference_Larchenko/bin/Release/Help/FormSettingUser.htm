Вікно налаштування користувачів

Данне вікно призначене для облку даних про користувачів. Перехід до налаштування користувачів не залежить від інших налаштувань.

З лівої сторони вікна розташована таблиця для зручного відображення існуючих користувачів. З правої сторони розташовані написи та поля для перегляду або вводу даних. Перемикач з написом «Права користувача» відповідає за надання або зняття з користувача прав адміністратора.

Для переміщення по записам використовується навігатор з функціональними кнопками, що знаходиться нижче таблиці, або можна обрати потрібний рядок у таблиці за допомогою миші. Для додавання, редагування та видалення даних використовуються функціональні кнопки з відповідними підписами.
