Вікно введення даних про студента

Данне вікно необхідне для вводу даних про студента.

Для початку створення довідки необхідно натиснути на головному вікні кнопку «Створити довідку». Вікно, що відкрилося надає можливість ввести дані про студента, а саме: ПІБ англійською та українською мовами, дата народження, місце народження англійською та українською мовами, назва відділення, код відділення, назва спеціалізації, поточний номер семестру, дата початку та дата завершення навчання, причина відрахування і номер наказу.

Після заповнення даних про студента можна зберегти довідку натиснувши кнопку «Зберегти та вийти» для продовження створення довідки в інший час. Також є можливість переглянути документ вже на цьому етапі, для цього потрібно натиснути кнопку «Попередньо переглянути». Буде відкритий додаток Microsoft Word з частково заповненною довідкою. Після перегляду необхідно закрити додаток Word без збереження даних.

Для переходу у вікно вибору навчальних дисциплін, необхідно заповнити всі поля і натиснути кнопку «Продовжити».
