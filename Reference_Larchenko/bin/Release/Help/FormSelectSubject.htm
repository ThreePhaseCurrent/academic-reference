Вікно вибору навчальних дисциплін

Данне вікно слугує для вибору навчальних дисциплін.

З лівої сторони вікна розташовані перемикачі для вибору потрібного курсу та семестру. По центру розташований динамічний список існуючих навчальних дисциплін для кожного семестру. Обрати дисципліну можна натисканням лівої кнопки миші по потрібному рядку. Передбачений швидкий вибір або зняття з вибору всіх дисциплін кнопками «Обрати все» і «Зняти все» відповідно. З правої сторони вікна розміщене візуальне відображення кількості обраних дисциплін для кожного семестру.

Для кожного семестру необхідно обрати хоча б одну дисципліну.

Для переходу до вікна виставлення оцінок, необхідно обрати для кожного семестру хоча б один предмет та натиснути кнопку «Продовжити».
