Головне вікно

Дане вікно є головним у програмі.

В лівому верхньому кутку розташована інформація про поточний обліковий запис. Для зміни облікового запису (повернутися в вікно авторизації) потрібно натиснути кнопку "Вийти".

Якщо Ви користуєтесь програмою вперше, то для початку роботи програму необхідно налаштувати. Потрібно перейти в налаштування програми натиснувши кнопку "Налаштування довідки". Для входу в налаштування обліковий запис повинен мати права адміністратора. Детальніше процес налаштування описаний в розділі "Налаштування програми".

Для початку створення академічної довідки необхідно натиснути кнопку "Створити довідку".

Для перегляду збережених довідок необхідно натиснути кнопку "Відкрити довідку".
