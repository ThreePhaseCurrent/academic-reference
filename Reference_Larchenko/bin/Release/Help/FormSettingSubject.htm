Вікно налаштування навчальних дисциплін

Данне вікно призначене для облку даних про навчальні дисципліни. Перехід до налаштування навчальних дисциплін відбувається після налаштування спеціалізацій.

З лівої сторони віна розташована таблиця для відображення даних, з правої сторони розташовані перемикачі та списки для вибору потрібного відділення, спеціальності, спеціалізації, номеру курсу та семестру. Нижче перемикачів розташовані написи та поля для вводу або редагування даних.

Для переміщення по записам використовується навігатор з функціональними кнопками, що знаходиться нижче таблиці, або можна обрати потрібний рядок у таблиці за допомогою миші. Для додавання, редагування та видалення даних використовуються функціональні кнопки з відповідними підписами.

Після завершення налаштування даних про навчальні дисципліни, програма є готовою для роботи, але крім цього програма має необов'язкові налаштування.
