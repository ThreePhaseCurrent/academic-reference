Вікно налаштування відділень

Данне вікно призначене для обліку данних про відділення. Перше налаштування програми починається саме з цього вікна.

З лівої сторони вікна розташована таблиця для відображення існуючих даних. З правої сторони розташовані поля з написами для зручного перегляду обраних даних та їх зміну, редагування або додавання.

Для переміщення по записам використовується навігатор з функціональними кнопками, що знаходиться нижче таблиці, або можна обрати потрібний рядок у таблиці за допомогою миші.

Для додавання, редагування та видалення даних використовуються функціональні кнопки з відповідними підписами.
