Вікно відкриття збережених довідок

Вікно слугує для перегляду та відкриття збережених довідок.

З лівої сторони вікна розташована таблиця для відображення існуючих довідок, з правої сторони розташовані текстові поля з написами для зручного перегляду та введення даних.

Для переміщення по таблиці використовується навігатор з функціональними кнопками, або можна обрати потрібний рядок у таблиці за допомогою миші. Для видалення непотрібної довідки використовується кнопка «Видалити». Для перегляду власної папки з створеними довідками потрібно натиснути кнопку «Відкрити папку».

Щоб відкрити обрану довідку необхідно натиснути кнопку «Відкрити довідку», всі збережені дані будуть введені у відповідні поля у вікні введення даних про студента для продовження створення довідки.
