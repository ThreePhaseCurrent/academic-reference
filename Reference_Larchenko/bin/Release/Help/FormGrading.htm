Вікно оцінювання

Вікно призначене для виставлення оцінок навчальним дициплінам.

З лівої сторони розташована таблиця для відображення обраних навчальних дисциплін, з правої сторони розташовані текстові поля, списки, перемикачі для зручного відображення дисциплін та виставлення оцінок. Для зміни шкали оцінювання використовується випадному список «Шкала оцінювання». Тип оцінки можна змінити обравши потрібний у випадному списку «Тип оцінки». Передбачено швидку зміну кількості годин для кожної дисципліни у відповідному текстовому полі. Для виставлення оцінки використовується перемикач з написом «Оцінка».

Для переходу до створення довідки повинні бути виставлені оцінки для всіх дисциплін (оцінка вища за нуль).

Для завершення створення довідки необхідно виставити оцінки для всіх дисциплін та натиснути кнопку «Створити довідку». Програма запропонує дати назву для довідки та надасть можливість зберегти документ на жорсткому диску. Після збереження довідки вона буде відразу відкрита для остаточного перегляду.
