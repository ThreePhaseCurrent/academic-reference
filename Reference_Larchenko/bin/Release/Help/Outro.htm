Вікно авторизації

Данне вікно слугує для входу користувача в власний обліковий запис. Щоб виконати вхід в існуючий обліковий запис необхідно ввести логін та пароль, які були створені при реєстрації.

Якщо користувач не має свого облікового запису - пропонуєтсья пройти реєстрацію натиснувщи кнопку "Зареєструватись". З'явиться вікно з можливістю створити новий обліковий запис.

Для виконання реєстрації необхідно ввести дані про користувача, а саме: ПІБ, логін та пароль. Після завершення вводу необхідно натиснути кнопку "Зареєструватись".

Якщо всі поля заповнені та такого ж самого користувача не було зареєстровано раніше - реєстрація буде виконана успішно. В інших випадках програма виведе інформацію про помилку з описанням.
