Вимоги до програмно-апаратного комплексу:

  • процесор Pentium з тактовою частотою не менше 1 ГГц;
  • об'єм оперативної пам'яті не менше 1 ГБ
  • вільного місця на жорсткому диску не менше 420 МБ
  • відеокарта і монітор, що підтримують режим Super VGA з роздільною здатністю не менше, ніж 1024x768 пікселів
  • принтер з відповідними драйверами для друку документів

Вимоги до програмного забезпечення:

  • операційна система Windows XP, або новіші версії
  • офісний пакет Microsoft Office, а саме текстовий редактор Microsoft Word 2010;
  • програмне забезпечення Framework 4.0
