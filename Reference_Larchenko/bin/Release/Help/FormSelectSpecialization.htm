Вікно налаштування спеціалізацій

Данне вікно призначене для облку даних про спеціалізації. Перехід до налаштування спеціалізацій відбувається після налаштування спеціальностей.

З лівої сторони віна розташована таблиця для відображення даних, з правої сторони розташовані перемикачі та списки для вибору потрібного відділення та спеціальності. Нижче перемикачів розташовані написи та поля для вводу або редагування даних.

Для переміщення по записам використовується навігатор з функціональними кнопками, який розташований нижче таблиці, або можна обрати потрібний рядок у таблиці за допомогою миші. Для додавання, редагування та видалення даних використовуються функціональні кнопки з відповідними підписами.

Після завершення налаштувань спеціалізацій потрібно повернутися на попереднє вікно та виконати налаштування навчальних дисциплін у відповідному вікні натиснувши кнопку «Редагувати дисципліни».
