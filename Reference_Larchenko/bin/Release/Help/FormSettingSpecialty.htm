Вікно налаштування спеціальностей

Данне вікно призначене для облку даних про спеціальності. Перехід до налаштування спеціальностей відбувається після налаштувань відділень.

З лівої сторони вікна розташовується таблиця для відображення даних, з лівої сторони розташовані написи та поля для вводу або перегляду інформації. Також розташований комбінований список з відділеннями, що існують. Відповідно кожне відділення має свої унікальні спеціальності.

Для переміщення по записам використовується навігатор з функціональними кнопками, що знаходиться нижче таблиці, або можна обрати потрібний рядок у таблиці за допомогою миші. Для додавання, редагування та видалення даних використовуються функціональні кнопки з відповідними підписами. Наступним кроком є налаштування спеціалізацій
